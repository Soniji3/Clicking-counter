<h1> Click Counter <a href="https://soniji3.github.io/Clicking-counter/">Preview</a>  </h1>
<p>
  A simple and interactive click counter web application built using HTML, CSS, and JavaScript. 
  The project allows users to increase, decrease, and reset the counter with a clean and responsive user interface. 
  It helps beginners understand DOM manipulation, event handling, and basic JavaScript functionality.
</p>

<h2>Features</h2>
<ul>
  <li>Increase counter value</li>
  <li>Decrease counter value</li>
  <li>Reset counter instantly</li>
  <li>Responsive and user-friendly design</li>
  <li>Beginner-friendly JavaScript project</li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>

<h2>Purpose</h2>
<p>
  This project was created to practice core front-end development concepts such as 
  DOM manipulation, event listeners, functions, and UI styling.
</p>

<h2>How to Run</h2>
<ol>
  <li>Download or clone the repository</li>
  <li>Open the <code>index.html</code> file in your browser</li>
  <li>Start using the click counter</li>
</ol>
